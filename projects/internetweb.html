Internet and Web

Internet

The internet is a network for the whole world and it is connected between devices letting user to do communication and also data sharing by using TCP/IP.

Web

The web is world wide web in long term and it is a way to connect in transfering documents and resources by using browsers with HTTP/HTTPS.

The different between Internet and Web

The internet is the whole infrastructure of a network of networks and The web is kind of like a service that needs the internet to go inside websites and content that you are looking for.

ASCII and Unicode

ASCII is a character encoding standard which use 7 bits to represent all of the 128 characters and unicode is a universal encoding standard ,it is supporting characters from all the languages and scripts.

What is UTF-8?

UTF-8 is a encoding for unicode that uses 1 to 4 bytes per character, it is compatible with ASCII and also efficient for many other languages.

URL

URL is uniform resource locator and it specifies the address of a resource on the Internet(e.g. https://www.example.com).

DNS

DNS is domain name system and it translates on domain names (e.g., amazon.com) into IP addresses by making the Web user-friendly.

W3C Recommendations

It is for the world wide web Consortium (W3C) for building and maintaining the Web (e.g., HTML, CSS).

XML

I think XML is extensible markup language structures and it can also stores data in human-readable format.

XHTML

XHTML is hypertext markup language,It is like the standard language that use for creating and displaying web pages.

HTML5

The latest version of HTML, it is supporting multimedia and has responsive design and use for modern web applications.
