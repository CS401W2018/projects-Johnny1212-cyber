What is Normal Flow in HTML?

Normal flow in HTML refers to the default layout behavior of elements on a web page. Block elements (e.g., <div>, <p>, <h1>-<h6>) stack vertically, while inline elements (e.g., <span>, <a>, <strong>) align horizontally within a block.

What Does Float Do? Why Do You Need to Clear the Float?

The float is like the older and younger sibling; they work in order and share space. In order to work, the older sibling must give up space for the younger sibling to work.

We need to clear the float because the remaining part of the HTML keeps following whatever has been floated.

What Do Absolute, Fixed, and Relative Do?

  • Relative: Allows an element to slide from its original position.
  • Absolute: Moves an element over the page from the top-right corner and positions it from that point, e.g., 50px from the top and 40px from the right. Often used for headers and can be scrolled.
  • Fixed: Stays in place even when scrolling; it remains in the browser window.
