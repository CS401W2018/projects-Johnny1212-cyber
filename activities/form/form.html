What are different ways to style your form?

You can style a form using CSS with different methods like:

  • CSS classes and IDs
  • Inline styles
  • Flex and Grid for layout
  • Bootstrap or other frameworks

Why does the professor like to use flex?

Flexbox is used because it makes form layout easier by aligning elements easier, allowing responsiveness and proper spacing for mobile.
