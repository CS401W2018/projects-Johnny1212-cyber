What is JavaScript?

JavaScript is the most well-known programming language today.

How did JavaScript get its name?

In the late 1990s, two companies, Microsoft and Sun Microsystems, were competing. They created the Java programming language to improve web browsers. Since Sun Microsystems owned Java, they named it JavaScript.

What are the two ways you can run JavaScript?

  • The first way is running JavaScript in the DOM (Document Object Model).
  • The second way is using Node.js (Express.js).
