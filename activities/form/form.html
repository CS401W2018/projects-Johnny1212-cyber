What is AJAX?

AJAX (Asynchronous JavaScript and XML) is a web development technique used to create fast and dynamic web applications. It allows web pages to update content without refreshing the entire page by making asynchronous requests to a server.

Why do you put your JS code in a separate file?

Placing JavaScript in a separate file makes the code easier to maintain as it keeps HTML clean and organized. It also improves performance because browsers can cache the external script, reducing load times. Using separate files allows for code reusability across multiple webpages. Additionally, it enhances security.
