Questions & Answers

Why is the viewport meta tag important?

The viewport meta tag makes web pages responsive by adjusting to different screen sizes, to have proper scaling on mobile devices.

How can you adjust the CSS for printing?

Create another CSS file for printing and link it to HTML. Add media="screen" for normal styles and media="print" for print-specific styles.

When would you want to adjust the CSS for printing?

To make the page look good when printed.
