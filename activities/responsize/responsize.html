Questions & Answers

Why is the viewport meta tag important?

The viewport meta tag helps websites look good on all screen sizes, especially on phones. Without it, pages might be too small or not fit properly.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

If you set flex-direction twice for the same thing, the last one in the CSS file will take priority because the browser follows the most recent rule.

How do you change the CSS for a web page displayed on a phone?

To change CSS for phones, use media queries. For example, @media ONLY screen AND (max-width: 600px) applies special styles when the screen is 600 pixels or smaller, making the site look better on small screens.
