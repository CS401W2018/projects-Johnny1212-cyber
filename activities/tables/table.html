When to Use a Table?

Use a table when you need to structure data, such as in a spreadsheet or a schedule.

When Not to Use a Table?

Avoid using tables for page layout as they can be hard to display properly on phones.

Table Accessibility

For accessibility, use <th> for column names, <thead> for labels, <tbody> for data, and <tfoot> for summaries.
